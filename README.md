# CPRF

Сервис интеллектуальной системы прогнозирования болезней системы кровообращения у населения высокоурбанизированного региона.
Подготовлен в рамках финала хакатона Цифровой прорыв 2020.

### Реализовано на
  - Java Spring JPA
  - Vue.js

#### Frontend

```sh
$ cd views

$ npm run serve
```
